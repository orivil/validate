# Validator

Validator is a tool for validating post form values.

## Install

go get -v gopkg.in/orivil/validator.v0

## Test

```
cd $GOPATH/src/gopkg.in/orivil/validator.v0

go test example_test.go
```

## Usage

see example_test.go
